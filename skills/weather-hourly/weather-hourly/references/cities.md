## 使用自定义坐标

当城市不在表中时，通过以下方式获取经纬度：
1. 打开百度地图，右键点击目标位置 → 复制经纬度
2. 或访问：https://lbs.qq.com/getPoint/

使用方式：
```bash
python scripts/fetch_hourly.py --city 孝感 --lat 30.92 --lon 113.92 --date 2026-05-06
```
