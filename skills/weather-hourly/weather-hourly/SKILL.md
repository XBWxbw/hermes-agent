# Weather Hourly — 逐小时天气预报

基于 Open-Meteo（https://open-meteo.com），无需注册，无需 API Key，全球覆盖，支持最多 16 天预报。

## 快速流程

1. **解析用户意图** — 提取目标城市 + 目标日期
2. **映射经纬度** — 从内置城市表查找，或让用户提供坐标
3. **调用 API** — 执行 `scripts/fetch_hourly.py`
4. **格式化输出** — 解析结果，以表格形式展示逐小时数据
5. **生成报告** — 归档到 `Prime\Guides\` 或用户指定路径（若用户要求）

## 执行脚本

```bash
python scripts/fetch_hourly.py --city 武汉 --date 2026-05-06
python scripts/fetch_hourly.py --lat 30.52 --lon 114.31 --city 武汉 --date 2026-05-06
python scripts/fetch_hourly.py --city 北京 --date tomorrow
python scripts/fetch_hourly.py --city 深圳,武汉,南京 --date 2026-05-07
```

参数说明：
- `--city`：城市名（支持逗号分隔多城市）
- `--date`：日期，格式 `YYYY-MM-DD`，或 `today` / `tomorrow`
- `--lat` / `--lon`：自定义经纬度（优先级高于城市名查表）
- `--fields`：可选，指定返回字段（默认：temp,weather,rain_prob,precip,wind,humidity）

## 输出格式

脚本默认输出 Markdown 格式表格，可直接嵌入报告。若用户要求保存文件，输出为 `.md` 文件归档到 Prime 工作目录。

## 数据说明

| 字段 | 含义 | 单位 |
|------|------|------|
| 气温 | 2米高度实时气温 | °C |
| 天气 | WMO 代码映射中文描述 | — |
| 降雨概率 | 该小时降水概率 | % |
| 降水量 | 该小时实际降水量 | mm |
| 风速 | 10米高度风速 | km/h |
| 湿度 | 相对湿度 | % |

## 日期范围限制

- 最多查询今天起 **16 天内**的数据
- 超出范围时提示用户并建议查询历史天气 API
